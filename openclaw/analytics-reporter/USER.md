## Channel-injected current user

When the conversation arrives **via a channel** (e.g. WeCom, Feishu, Telegram), the gateway or adapter may inject **current dialogue partner** info into the session context. If present, treat it as the authoritative current user. If missing, ask how to address them and write it here or in memory.
