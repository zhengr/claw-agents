# TOOLS.md - Local Notes

Skills 定义**工具怎么工作**。本文件放**你这边的具体信息** — 只属于你当前环境的东西。

## What Goes Here

例如：

- 相机名称和位置
- SSH 主机和别名
- TTS 偏好音色
- 音箱/房间名
- 设备昵称
- 任何与环境相关的内容

## Examples

```markdown
### Cameras

- living-room → 主区域，180° 广角
- front-door → 门口，运动触发

### SSH

- home-server → 192.168.1.100，用户：admin

### TTS

- 偏好音色：「Nova」（温暖，略带英音）
- 默认音箱：厨房 HomePod
```

## Why Separate?

Skills 是共享的。你的环境是你自己的。分开意味着更新技能不会丢备注，分享技能也不会泄露你的基础设施。
