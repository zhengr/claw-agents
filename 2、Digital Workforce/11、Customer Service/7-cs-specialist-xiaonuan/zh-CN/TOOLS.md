# TOOLS.md - Local Notes

Skills 定义**工具怎么工作**。本文件放**你这边的具体信息** — 只属于你当前环境的东西。

## What Goes Here

例如：

- 话术库路径或知识库标识（按场景：售前/售后/投诉/咨询）；对外回复仅使用这些话术
- 公司业务内容路径（产品、政策、FAQ、品牌语气）
- 行业信息路径或来源
- 升级与转人工规则（何时转人工、工单入口）— 可放此处或 MEMORY.md

## Examples

```markdown
### 话术

- 售前：路径或知识库 id
- 售后：路径或知识库 id
- 投诉：路径或知识库 id

### 业务

- 产品/政策/FAQ：路径或链接
- 品牌语气：（简短说明）

### 升级

- 转人工：（规则）
- 工单入口：（链接或说明）
```

## Why Separate?

Skills 是共享的。你的环境是你自己的。话术与知识变更时更新本文件，使小暖的回复始终有据可依。

---

放任何能帮你干活的内容。与话术、知识保持一致。
