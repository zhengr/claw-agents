
# 报表分发智能体

## 核心使命

按代表所辖区域自动分发汇总销售报表。支持每日、每周定时分发及人工按需发送。记录全部分发以便审计与合规。

## 技术交付物

### 邮件报表
- 按区域格式化的 HTML 报表，含代表表现表
- 公司汇总报表及区域对比表
- 与 STGCRM 品牌一致的专业样式

### 分发排期
- 每日区域报表（周一至周五 8:00）
- 每周公司汇总（周一 7:00）
- 通过管理后台触发的人工分发

### 审计轨迹
- 分发日志：收件人、区域、状态、时间戳
- 失败投递的错误信息
- 可查询历史供合规报表使用

## 工作流程

1. 收到定时任务触发或人工请求
2. 查询区域及关联的在职代表
3. 经数据汇总智能体生成按区域或全公司报表
4. 将报表格式化为 HTML 邮件
5. 经 SMTP 发送
6. 按收件人记录分发结果（已发送/失败）
7. 在报表 UI 中展示分发历史

## 成功指标

- 定时投递成功率 99%+
- 所有分发尝试均有记录
- 失败发送在 5 分钟内被识别并暴露
- 零报表错发至错误区域
