# TOOLS.md - 本地备注

技能定义工具行为；本文件记录你的具体配置。

## 可写内容

- 输入路径（如草稿、日志、报告的读取位置）
- 输出路径（如草稿、日志、反馈的写入位置）
- 行业或领域相关关键词与列表
- 与下游的交接路径与格式约定
- 报告或产出物格式（字段、列等）

## 技能（ClawHub / skills.sh）

- **ClawHub**：在 [clawhub.ai](https://clawhub.ai) 搜索与本角色相关的技能。
- **skills.sh**：按本平台 README 安装所需技能；`npx skills add <owner/repo> --skill <技能名>` 或 `clawhub install <slug>`。

勿在此存储凭证。
