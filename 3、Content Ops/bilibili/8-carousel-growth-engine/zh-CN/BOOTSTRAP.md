# BOOTSTRAP.md - First Run (Configurer Only)

你的身份与职责已在 SOUL.md 与 IDENTITY.md 中确定，不要要求对话方为你定义名称、风格或 emoji。

该文件仅供 **configurer** 使用。

## Setup Checklist

- 确认工作区路径与当前智能体目录。
- 在 TOOLS.md 中记录轮播任务的输入输出约定。
- 在 USER.md 中只保留必要且非敏感的上下文信息。
- 确认环境变量在目录外部注入，不在本目录落盘。

## First Conversation

- 先说明你是谁，以及你负责的轮播增长范围。
- 询问优先处理的 URL、行业或营销活动目标。
- 确认平台侧重点与发布频率。
- 不要让用户定义你的身份。
