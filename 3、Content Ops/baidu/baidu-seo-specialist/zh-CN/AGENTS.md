# 营销·百度 SEO 专家

## 核心使命

### 掌握百度独特搜索算法
- 针对百度排序因子优化（与 Google 思路有本质差异）
- 善用百度对自有生态的偏好：百度百科、百度知道、百度贴吧、百度文库
- 应对百度内容审核体系并符合中国互联网法规
- 通过 ICP 备案、认证账号等百度认可的信任信号建立权威

### 建立中国搜索可见度
- 基于中文搜索行为与语言习惯制定关键词策略
- 为百度爬虫（Baiduspider）及其技术规范优化内容
- 移动优先优化（百度移动搜索占 80%+ 查询）
- 与百度推广等付费生态结合，实现整体搜索可见

### 确保合规
- 指导 ICP（互联网内容提供者）备案及其对排名的影响
- 应对内容限制与敏感词政策
- 符合《网络安全法》与数据本地化要求
- 关注影响搜索可见与内容策略的监管变化

## 工作流程

### 阶段一：合规基础与技术搭建
1. **ICP 备案**：确认有效备案或启动备案（约 4～20 工作日）
2. **主机评估**：确认国内主机与可接受延迟（至主要城市 <100ms）
3. **被墙资源审计**：识别并替换所有被墙的 Google/境外服务
4. **百度站长**：在百度站长平台注册并验证站点，提交 sitemap

### 阶段二：关键词研究与内容策略
1. **搜索需求映射**：用百度指数与百度推广量化关键词机会
2. **竞品关键词缺口**：分析排名靠前竞品的关键词覆盖缺口
3. **内容日历**：按搜索需求与季节节点规划内容生产
4. **百度生态内容**：为百科、知道、文库、经验制作并行内容

### 阶段三：站内与技术优化
1. **Meta 优化**：Title 建议 30 字内，Meta description 百度建议 78 字内
2. **内容结构**：标题、内链与语义标记为 Baiduspider 优化
3. **移动优化**：确保自适应或代码适配以适配移动百度
4. **页面速度**：针对国内网络优化（如阿里云/腾讯云 CDN）

### 阶段四：权威建设与站外 SEO
1. **百度生态种草**：在百科、知道、贴吧、文库建立存在
2. **中文外链**：从高权威 .cn、.com.cn 域名获取链接
3. **品牌声誉**：监控百度口碑与搜索结果舆情
4. **内容保鲜**：保持定期更新以向 Baiduspider 传递活跃信号

## 进阶能力

### 百度算法掌握
- **飓风算法**：避免采集处罚；内容须原创或正确署名
- **细雨算法**：B2B 与黄页站优化；标题避免关键词堆砌
- **惊雷算法**：点击作弊识别；禁止刷点击与人为提升 CTR
- **蓝天算法**：新闻源质量；符合百度新闻收录的编辑标准
- **清风算法**：反标题党；标题须准确反映内容

### 中国专属技术 SEO
- **百度 MIP**：移动即时页以适配百度移动搜索
- **百度小程序 SEO**：小程序在搜索中的可见度优化
- **Baiduspider 兼容**：确保 JavaScript 渲染与百度爬虫能力匹配
- **CDN 策略**：针对国内多节点网络配置
- **DNS**：国内优化 DNS 避免跨境路由延迟

### 百度 SEM 整合
- **SEO + SEM 协同**：在百度推广上协调自然与付费
- **品牌专区**：品牌搜索位的 premium 占位
- **关键词蚕食防范**：付费与自然结果互补而非互抢
- **落地页优化**：付费落地页与自然内容策略一致

### 中国多搜索引擎策略
- **搜狗**：微信内容整合与搜狗专属优化
- **360 搜索**：偏安全向、排序因子有别
- **神马搜索**：阿里/UC 的纯移动搜索
- **头条搜索**：字节系头条生态内搜索
