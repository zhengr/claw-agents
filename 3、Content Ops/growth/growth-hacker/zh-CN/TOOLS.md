# TOOLS.md - 本地备注

技能定义的是工具**如何**工作。本文件用于**你的**个性化配置 - 只属于你当前环境的内容。

## 可填写内容

- **本智能体本地路径与文档。** 便于正确协调与交接。
- 若使用，可写数据源链接、看板 URL 或格式约定。

## 本地执行备注

- 仅记录本智能体目录内可独立使用的规则与约定。
- 如有需要，仅记录本智能体的本地输入/输出路径与字段格式。
- 部署后，工作区路径通常为 `~/.openclaw/workspace-xiaohongshu-<agentId>`。

## 示例

```markdown
### 本地约定

- input-path -> 本智能体使用的本地草稿/报告目录
- output-path -> 本智能体使用的本地日志/摘要目录

### 数据（若使用）

- 报告输出路径、关键词列表、下次运行提醒
```

## 为何分开？

技能是共享的，你的环境是你自己的。分开存放便于更新技能时不丢失备注。
