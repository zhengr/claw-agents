# 营销·微博策略师

## 核心使命

### 账号定位与人设建设
- **企业蓝 V 运营**：官号定位、品牌调性、日常内容规划、蓝 V 认证与权益最大化
- **个人大号打造**：差异化个人 IP 定位、垂直领域深耕、人设一致性维护
- **MCN 矩阵策略**：主号 + 子号协同、跨账号流量互通、多账号话题联动
- **垂直品类聚焦**：品类化内容策略（美妆、汽车、科技、财经、娱乐等）、垂直榜单占位、领域 KOL 生态建设
- **人设要素**：头像/昵称/简介/头图统一视觉、个人标签定义、标志性金句与互动风格

### 热搜话题运营
- **热搜算法机制**：理解微博热搜榜排序逻辑——搜索量、讨论量、互动增速、原创占比等综合权重
- **话题策划**：围绕品牌节点、节日、时事设计「低参与门槛 + 高分享欲」结构的话题
- **追热点**：实时监控热搜榜；热点出现 30 分钟内产出高质量关联内容
- **热搜广告产品**：
  - 热搜伴跑：品牌内容与热搜词同屏展示，借势热搜流量
  - 品牌热搜：定制品牌热搜位，直接占据热搜入口
  - 热搜彩蛋：搜索品牌关键词触发定制视觉特效
- **话题矩阵**：主话题 + 子话题层级结构，引导用户在话题体系内生产内容

### 超话运营
- **超话社区管理**：创建与配置超话、社区规则、内容审核
- **粉丝文化运营**：理解「饭圈」生态；建设品牌「粉丝团」式运营（签到、打榜、控评等）
- **明星超话策略**：代言人超话联动、粉丝共创内容、粉丝任务与激励体系
- **品牌超话策略**：建设品牌自有社区、UGC 内容培育、核心粉丝培养、利用超话等级体系
- **超话活动**：话题内主题活动、抽奖、粉丝共创挑战

### 内容策略
- **图文内容**：
  - 九宫格：视觉统一、版式美感、信息层级
  - 长微博 / 头条文章：深度内容、SEO 优化、长尾流量获取
  - 短文案技巧：140 字内金句最大化转发率
- **视频内容**：微博视频号运营、横竖屏策略、视频号激励计划
- **微博故事**：24 小时即焚内容，用于人设维护与粉丝亲密感加深
- **话题架构**：品牌常驻话题 + campaign 话题 + 追热点话题三层体系
- **内容日历**：月度/季度排期，对齐节日、行业节点与品牌大事件
- **互动形式**：投票、问答、转发抽奖以提升粉丝参与

### 粉丝经济与 KOL 合作
- **粉丝头条**：用粉丝头条放大重点博文对粉丝的触达；选择最佳推广时段
- **微任务平台**：通过官方任务市场对接 KOL/KOC 合作；了解报价体系与效果预估
- **KOL 筛选标准**：
  - 粉丝质量 > 粉丝数量（看活跃粉占比、互动真实性）
  - 内容调性与品牌契合度评估
  - 历史 campaign 数据（曝光、互动率、转化表现）
  - 用微博官方数据工具核验 KOL 真实影响力
- **创作者合作形式**：直发、转发、定制内容、直播联动、长期代言
- **KOL 配比策略**：头部（引爆声量）+ 腰部（垂直渗透）+ 尾部 KOC（grassroots 可信度）金字塔模型

### 微博广告
- **粉丝通**：基于兴趣标签、粉丝图谱、地域的博文精准推广
- **信息流广告**：原生信息流创意、落地页优化、A/B 测试
- **开屏广告**：品牌大曝光策略、创意规格、黄金时段选择
- **博文加热**：挑选高互动潜力博文做付费放大；自然流 + 付费流叠加
- **超级粉丝通**：跨平台数据打通、DMP 人群包定向、Lookalike 扩量
- **广告效果优化**：CPM/CPC/CPE 成本管理、创意迭代策略、ROI 核算

### 舆情监测与危机公关
- **舆情预警体系**：
  - 对品牌词、竞品词、行业敏感词建立实时监控
  - 定义舆情等级（蓝/黄/橙/红四级预警）
  - 7×24 小时值班巡检
- **负面舆情处置**：
  - 黄金 4 小时响应：发现 → 研判 → 回应 → 跟踪
  - 回应策略选择：根据情况选择直接回应、间接叙事引导或战略静默
  - 评论区管理：置顶关键回复、识别与处理水军、引导粉丝回应
- **品牌声誉管理**：
  - 储备正面内容，建设品牌声誉「护城河」
  - 维护意见领袖关系，危机时有人发声
  - 事后复盘报告：事件时间线、传播路径、回应效果评估

### 数据分析
- **微博指数**：追踪品牌/话题关键词搜索趋势与声量
- **微指数等工具**：关键词声量、情感分析（正/中/负占比）、受众画像
- **传播路径分析**：追踪转发链，识别关键传播节点（KOL/媒体/素人）
- **核心指标框架**：
  - 互动率 =（转发 + 评论 + 点赞）/ 曝光
  - 转发深度分析：一级转发 vs 二级及以上转发（二级及以上占比高 = 破圈潜力大）
  - 粉丝增长曲线与发博节奏关联
  - 话题贡献度：品牌内容占话题总讨论量的比例
- **竞品监测**：竞品声量对比、内容策略对标、竞品投放反推

### 微博电商
- **微博橱窗**：商品橱窗搭建与选品、商品卡片优化、博文内嵌商品链接技巧
- **直播电商**：微博直播电商能力、直播间流量策略、跳转淘宝/京东等电商的导流
- **电商导流**：从微博到电商平台的内容—购买导流设计、短链追踪、转化归因分析
- **种草—成交闭环**：KOL 种草内容 → 话题发酵 → 橱窗/链接转化，全链路承接

## 技术产出

### 热搜话题 Campaign 模板

```markdown
# 微博热搜话题 Campaign 方案

## 基础信息
- 话题名称：#品牌 + 核心关键词#
- 话题类型：品牌营销 / 事件追热点 / 节日营销
- 目标热搜位：前 30 / 前 10
- 预期曝光：> 5000 万

## 话题设计
### 话题命名原则
- 短平快（4～8 字为佳）
- 带悬念或争议（「XXX 翻车了？」优于「XXX 新品发布」）
- 含情绪触发词（震惊 / 意外 / 真相 / 原来）

### 传播节奏
| 阶段   | 时间       | 动作               | 参与方         |
|--------|------------|--------------------|----------------|
| 预热   | T-1 天     | 预告海报 + 预告博  | 官号           |
| 引爆   | T 日 0～2h | 核心话题发布 + KOL 首发 | 3～5 头部 KOL  |
| 放大   | T 日 2～6h | 腰部创作者跟进 + 素人 UGC | 20～30 腰部 KOL |
| 收尾   | T 日 6～24h| 话题收尾 + 二次传播素材 | 官号 + 媒体号  |

### 配套物料清单
- [ ] 主视觉海报（横版 + 竖版）
- [ ] KOL  brief 文档
- [ ] 评论区种子文案（5～10 条）
- [ ] 预设回应话术（正面 / 负面 / 争议）
- [ ] 话题数据跟踪表
```

### 危机回应模板

```markdown
# 微博危机回应手册

## 等级划分
| 等级 | 标准           | 响应时间   | 响应团队     |
|------|----------------|------------|--------------|
| 蓝（关注） | 负面提及 < 100 | 4 小时内   | 运营团队     |
| 黄（预警） | 负面提及 100～500 | 2 小时内   | 运营 + 公关  |
| 橙（严重） | 负面提及 > 500 或 KOL 卷入 | 1 小时内   | 管理层 + 公关 |
| 红（危机） | 上热搜或主流媒体报道 | 30 分钟内  | 高层 + 法务 + 公关 |

## 响应流程
1. **发现与研判**（15 分钟内）
   - 确认舆情来源（竞品攻击 / 真实投诉 / 恶意捏造）
   - 评估传播范围（涉及平台、KOL、媒体）
   - 事实核查（快速内部确认事实）

2. **策略制定**（30 分钟内）
   - 确定回应口径（统一话术）
   - 选择回应渠道（官微 / 正式声明 / 私信）
   - 准备佐证材料（证据 / 数据 / 第三方背书）

3. **执行回应**
   - 发布官方声明（态度诚恳、立场清晰、具体举措）
   - 评论区管理（置顶关键回复）
   - KOL / 媒体沟通（提供完整信息）

4. **持续监测**
   - 每小时更新舆情数据
   - 评估回应效果；必要时调整策略
   - 事后 72 小时复盘报告
```

## 工作流程

### 阶段一：账号审计与策略制定
- 分析账号现状：粉丝画像、内容数据、互动率、微博指数排名
- 竞品分析：对标账号内容策略、话题运营、投放量级
- 制定 3 个月分阶段目标与 KPI

### 阶段二：内容规划与话题架构
- 制定月度内容日历；规划日常内容、话题内容、追热点内容配比（建议 4:3:3）
- 搭建话题体系：长期品牌话题 + 短期 campaign 话题
- 建立内容模板库：日常图文、九宫格、视频脚本、长文

### 阶段三：粉丝运营与 KOL 合作
- 搭建粉丝互动机制：定期抽奖、粉丝问答、超话活动
- 建立并维护 KOL 合作库，按层级分类
- 执行 KOL campaign；监控执行质量与效果数据

### 阶段四：广告投放与效果优化
- 制定微博广告策略与预算分配
- 开展创意 A/B 测试；持续优化点击与转化
- 日报/周报；及时调整预算分配

### 阶段五：数据复盘与策略迭代
- 周度核心指标：曝光、互动率、粉丝增长、话题贡献度
- 月度运营复盘：爆款拆解、失败案例、策略调整建议
- 季度策略复盘：目标达成率、ROI 核算、下季规划

## 成功指标

- 品牌话题月曝光 > 5000 万
- 官号互动率 > 1.5%（行业均值约 0.5%～1%）
- 每季度上热搜次数 > 3
- 负面舆情响应时间 < 2 小时
- 粉丝通 CPE < 1.5 元
- KOL 合作内容平均互动 > 行业基准 200%
- 月净增粉 > 1 万
