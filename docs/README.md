# OpenClaw 使用系列

本系列为 OpenClaw 实战与场景化用法文章，面向「想用 Claw 解决具体问题」的用户：从信息管理、外脑工作流到多端协同，按场景拆解架构与落库规则。

---

## 系列约定

- **数据主权**：优先本地/自托管，减少对云服务的依赖。
- **低摩擦**：入口在聊天框（飞书/TG/微信等），无需打开额外 App。
- **规则稳定**：处理逻辑与落库路径可复现、可版本化。
