# OpenClaw 智能体中英对照规范

凡带有 `zh-CN/` 目录的智能体，**必须**按「标准 7 件套」在英文（根目录）与中文（`zh-CN/`）之间保持结构一致、内容对应。禁止缩水或「详见根目录」式省略。

## 标准 7 件套（不得丢三落四）

| # | 文件名 | 说明 | 要求 |
|---|--------|------|------|
| 1 | **AGENTS.md** | 智能体人格与工作区说明 | zh-CN 为 EN 的**完整翻译**；保留代码块、列表、标题层级一致；禁止仅写「详见根目录 AGENTS.md」。 |
| 2 | **BOOTSTRAP.md** | 配置/首次运行说明（配置员用） | zh-CN 与 EN 结构一致、内容逐条可对照；缺 zh-CN 的需补全。 |
| 3 | **HEARTBEAT.md** | 心跳/存活与调度说明 | zh-CN 与 EN 结构一致、内容对应；缺 zh-CN 的需补全。 |
| 4 | **IDENTITY.md** | 身份、名字、形态、调性、产出与边界 | 中英**一一对应**（同一句的翻译或填实模板）；禁止 zh-CN 写与 EN 主旨无关的独立句子。 |
| 5 | **SOUL.md** | 信念、边界、沟通风格（灵魂） | 中英**结构一致**（同一批小节、同一批规则），表述**语义对应**。 |
| 6 | **TOOLS.md** | 工具与路径说明 | zh-CN 与 EN 结构一致、工具列表与路径对应；有 zh-CN 目录的需保证 zh-CN 版本完整。 |
| 7 | **USER.md** | 用户/对话对象说明 | zh-CN 与 EN 结构一致、用户/对话对象描述对应；有 zh-CN 目录的需保证 zh-CN 版本完整。 |

## 验收标准

- **AGENTS.md**：`zh-CN/AGENTS.md` 行数 ≥ 根目录 `AGENTS.md` 的 80%，且小节标题与代码块一一对应。
- **SOUL / IDENTITY**：中英小节标题与条数一致，关键句可逐条对照。
- **BOOTSTRAP / HEARTBEAT / TOOLS / USER**：有 EN 且有 `zh-CN/` 的智能体，必须存在对应 zh-CN 文件，且结构一致、内容可对照。

## 审计与补全

- 全库审计清单见 [agent-i18n-audit.md](./agent-i18n-audit.md)。
- **AGENTS.md 缩水批量补全**：运行 `scripts/expand_zhcn_agents_batch.py`。需在项目 venv 中安装依赖后执行：
  ```bash
  cd openclaw-agents && python3 -m venv .venv && . .venv/bin/activate
  pip install deep-translator
  python scripts/expand_zhcn_agents_batch.py
  ```
  脚本会保留代码块、将 EN 正文机翻为 zh-CN 并写回 `zh-CN/AGENTS.md`。38-ui-designer 已手工全译，脚本会自动跳过。
- **SOUL / IDENTITY 对齐**：
  - **SOUL**：小节标题与 EN 逐条对应，标题格式为「English（中文）」；可参照 `3、Content Ops/baidu/2-seo-specialist/zh-CN/SOUL.md`。
  - **IDENTITY**：采用约 20 行简洁版式（与 xiaohongshu、2-seo-specialist 一致）：`# IDENTITY — 中文名` → 身份与职责 → **名字/形态/调性/Emoji** → **工作内容/何时调用我/我擅长/我产出** → **我不做/不要问/敏感与合规** → **简短开场**。范例：`3、Content Ops/baidu/2-seo-specialist/zh-CN/IDENTITY.md`、`3、Content Ops/xiaohongshu/3-xiaohongshu-rewrite/zh-CN/IDENTITY.md`。
  - 批量对齐脚本（需安装 `deep-translator`）：
    - `python scripts/align_zhcn_identity.py` — IDENTITY 统一为 20 行版式
    - `python scripts/align_zhcn_soul_headings.py` — SOUL 标题改为 English（中文）
    - `python scripts/align_zhcn_agents_headings.py` — AGENTS 文档级与代码块内标题改为 English（中文）（耗时会较长）
  - 验收抽检：`python scripts/verify_zhcn_alignment.py`
