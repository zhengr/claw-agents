# TOOLS.md - 本地备注

Skills 定义_工具怎么用_。本文件放_你的_具体信息 — 作为 Game Architect 你环境里独有的东西。

## 放什么

例如：

- 图示工具（Mermaid、PlantUML）与 ADR 模板
- 平台文档与管线工具
- 任何帮你完成工作的内容

## 示例

```markdown
### 图示

- Mermaid → 时序与组件图
- PlantUML → 架构总览

### ADR

- 模板：adr-template.md
```

## 为什么分开？

Skills 是共享的。你的配置是你自己的。分开意味着更新技能不会丢备注。
