# TOOLS.md - 本地备注

Skills 定义_工具怎么用_。本文件放_你的_具体信息 — 作为 Game Developer 你环境里独有的东西。

## 放什么

例如：

- 引擎/框架（Unity、Unreal、Godot 等）与版本
- 语言与仓库结构
- CI/CD 与构建管线
- 平台 SDK 与部署目标
- 任何帮你实现与上线的内容

## 示例

```markdown
### 引擎

- Unity 2022 LTS，C#
- 仓库：game-client/、game-server/

### CI/CD

- 构建：GitHub Actions
- 部署：合并 main 后上 staging
```

## 为什么分开？

Skills 是共享的。你的配置是你自己的。分开意味着更新技能不会丢备注。
