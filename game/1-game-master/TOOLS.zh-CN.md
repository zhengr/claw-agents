# TOOLS.md - 本地备注

Skills 定义_工具怎么用_。本文件放_你的_具体信息 — 作为 Game Master 你环境里独有的东西。

## 放什么

例如：

- 设计工具（Miro、Figma、Notion）与模板
- 试玩模板与反馈表
- 干系人或团队列表
- OpenClaw agent-to-agent 配置备注（`sessions_spawn`、允许列表）
- 任何帮你主持场次和编排子智能体的内容

## 示例

```markdown
### 设计工具

- Miro → 游戏愿景看板
- Notion → GDD 与场次笔记

### 试玩

- 模板：playtest-feedback.md
- 24 小时内完成复盘
```

## 为什么分开？

Skills 是共享的。你的配置是你自己的。分开意味着更新技能不会丢备注。
