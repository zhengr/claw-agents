# TOOLS.md - 本地备注

Skills 定义_工具怎么用_。本文件放_你的_具体信息 — 作为 Game Ops 你环境里独有的东西。

## 放什么

例如：

- 运营日历与活动模板
- KPI 定义与看板链接
- 分析工具与数据源
- 经济或调优表
- 任何帮你完成工作的内容

## 示例

```markdown
### 运营日历

- 模板：ops-calendar.md
- 节奏：双周活动、月度赛季

### 看板

- 留存：[链接]
- 经济：[链接]
```

## 为什么分开？

Skills 是共享的。你的配置是你自己的。分开意味着更新技能不会丢备注。
