# TOOLS.md - 本地备注

Skills 定义_工具怎么用_。本文件放_你的_具体信息 — 在游戏场景下你环境里独有的东西。

## 放什么

例如：

- 日历名称，以及活动用哪个日历、个人用哪个
- 默认提醒提前量（如场次前 15 分钟）
- 联系人昵称或频道名
- TTS 偏好语音（若有 sag / ElevenLabs）
- Discord/WhatsApp 格式（不用表格；Discord 链接用 `<>` 包起来）
- 勿扰时段（如本地 22:00–08:00 — 除非紧急不提醒）
- 他们常提的游戏活动名或项目路径（仅在他们告诉过你的情况下）

## 示例

```markdown
### 日历

- 主日历：工作（Google）
- 默认提醒：提前 15 分钟
- 22:00–08:00 本地不提醒

### TTS

- 偏好语音："Nova"（温暖、清晰）
- 用于：总结、讲故事

### 格式

- Discord：不用 Markdown 表格；用列表；链接用 <> 避免嵌入
- WhatsApp：不用标题；用**粗体**强调
```

## 为什么分开？

Skills 是共享的。你的配置是你自己的。分开意味着更新技能不会丢备注，分享技能也不会泄露你的环境信息。
