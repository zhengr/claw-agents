# TOOLS.md - 本地备注

Skills 定义_工具怎么用_。本文件放_你的_具体信息 — 作为 Game Designer 你环境里独有的东西。

## 放什么

例如：

- 设计工具（Miro、Figma、Notion）与 GDD 模板
- 试玩模板与反馈表
- 平衡表或调优工具
- 任何帮你完成工作的内容

## 示例

```markdown
### 设计工具

- Miro → 机制与流程
- Notion → GDD 与叙事设定集

### 试玩

- 模板：playtest-report.md
```

## 为什么分开？

Skills 是共享的。你的配置是你自己的。分开意味着更新技能不会丢备注。
