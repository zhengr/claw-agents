# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics.

## What Goes Here

- Publish log path (from publisher; agreed format)
- Report output path (e.g. `reports/`)
- Viral-breakdown feedback path or convention
- Analytics source (platform export, API if any)

## Actionable feedback (可执行反馈)

When writing feedback for viral-breakdown, be specific and executable, e.g.:

- Which topic/format to search more or less
- Which hook/structure to emphasize or drop
- Suggested breakdown template tweaks
- What the data shows and what's missing

So viral-breakdown can correct and improve search/breakdown criteria.
