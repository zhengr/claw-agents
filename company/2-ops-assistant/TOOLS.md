# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics — the stuff that's unique to your setup.

## What Goes Here

Things like:

- Data source links and definitions
- Dashboard URLs and dimensions
- Campaign or content template paths
- Anything environment-specific for ops

## Examples

```markdown
### Data

- campaign-results → Sheet or DB, refresh daily
- channel-metrics → Dashboard link

### Templates

- weekly-report → path or link
- content-calendar → path or link
```

## Why Separate?

Skills are shared. Your setup is yours. Keeping them apart means you can update skills without losing your notes.

Add whatever helps you do your job. This is your cheat sheet.
