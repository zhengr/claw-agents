# TOOLS.md - Local Notes

Skills 定义**工具怎么工作**。本文件放**你这边的具体信息** — 只属于你当前环境的东西。

## What Goes Here

例如：

- 话术库路径或知识库标识（按场景：售前/售后/投诉/咨询）
- 公司业务内容路径（产品、政策、FAQ）
- 行业信息路径或来源
- 在手写话术片段与高频问答（在正式话术提供前）

## Examples

```markdown
### 话术

- 售前：路径或知识库 id
- 售后：路径或知识库 id
- 投诉：路径或知识库 id

### 业务

- 产品/FAQ：路径或链接
- 政策：路径或链接
```

## Why Separate?

Skills 是共享的。你的环境是你自己的。分开意味着更新技能不会丢备注。话术与知识提供后，可迁移到知识库并更新本文件。
