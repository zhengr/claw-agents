# BOOTSTRAP.md - 首次运行（仅配置者）

你的**身份与职责**已在 SOUL.md 与 IDENTITY.md 中定义。**不要**询问对话方你的名字、形态、调性或 emoji — 已在 IDENTITY.md 中固定。

本文件仅供**配置者**使用（如填写 USER.md、路径）。尚无 memory；memory 文件在创建前不存在是正常的。

---

1. **环境与依赖：** 确认工作区路径与当前智能体目录；若使用外部技能或 API，在 TOOLS.md 中注明所需环境变量或配置（勿在 TOOLS.md 存密钥）；若读写特定路径，在 TOOLS.md 中列出。
2. **USER.md：** 创建或更新 USER.md，填写你在帮谁（姓名、称呼、时区等，仅使用对方主动提供或任务必需的信息）。不要问「该怎么称呼你」或「想怎么称呼我」。
3. **TOOLS.md 与路径：** 在 TOOLS.md 中写明输入/输出路径与格式约定。勿在本目录任何文件中存凭证、API 密钥或登录态。
4. **memory/ 与 MEMORY.md：** 无则创建 memory/；主会话可按设计使用 MEMORY.md。无需预填，智能体会在运行中写入。
5. **首次对话：** 先说明你是谁、能做什么（用 IDENTITY 中的开场），再问对方要完成什么或要配置什么。不要盘问对方。配置完成后**删除本文件**。
