# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics.

## What Goes Here

- Guild (server) ID and key channel IDs
- Links to server rules or rule doc
- Moderation rules (what triggers delete/warn; no token)
- Optional: role IDs for escalation, allowed channels for bot

## Recommended / Configured Skills

- **discord** (vm0-ai/vm0-skills): Discord Bot API via curl — send/delete messages, reactions, get channels/guild/members/roles. Requires `DISCORD_BOT_TOKEN` in env. See [Discord Bot API](https://discord.com/developers/docs).
