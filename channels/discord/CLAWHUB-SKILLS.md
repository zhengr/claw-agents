## 重点技能简介（Top 6）

### discord · steipete

通过 Clawdbot 的 discord 工具控制 Discord：发消息、加反应、贴/上传贴纸与表情、发起投票、管理线程/置顶/搜索、查询权限或成员/角色/频道信息、在 DM 或频道内执行 moderation 等。高下载量与 Stars，适合作为默认 Discord 能力。

### discord-voice · avatarneil

在 Discord 语音频道内与 Claude AI 进行实时语音对话，多版本迭代。

### discord-chat · bowenQT

使用 message 工具在 Discord 频道内发送消息、回复消息、搜索消息历史。适用于用户需要与 Discord 沟通（发/回/搜）、查看 Discord 动态或与频道互动时。

### discord-hub · codedao12

OpenClaw 专用，基于 Discord Bot API 的工作流：interactions、commands、messages 及各类操作，通过直接 HTTPS 请求实现。

### discord-admin · TheBigBrainChad

从 A 到 Z 的 Discord 服务器管理：频道/角色/成员管理、AutoMod、webhooks、模板、审计日志、定时活动、线程及完整服务器控制，通过 CLI 操作。

### discord-voice-deepgram · adriel1006

在 Discord 语音频道中使用 Deepgram 流式 STT 与低延迟 TTS 进行语音对话。

## 安装、更新与卸载

### 安装
- **搜索**：[ClawHub discord](https://clawhub.ai/skills?sort=downloads&q=discord) 查看最新技能与下载量。
- **命令**：`clawhub install <slug> --workdir ~/.openclaw;`
- 安装后技能目录名需与 openclaw 配置中 `agents.list[].skills` 一致。
- **Token**：使用前配置 `DISCORD_BOT_TOKEN`（[Discord Developer Portal](https://discord.com/developers/applications) 创建应用并添加 Bot），勿写入 TOOLS.md 或配置文件。
- **补充**：skills.sh 安装：`npx skills add <owner/repo> --skill <名>`；Discord 技能列表见本目录 [README.md](./README.md)。

### 更新
- 重新执行 `clawhub install <slug> --workdir ~/.openclaw;` 覆盖安装即可；具体以 ClawHub CLI 文档为准。

### 卸载
- **命令**：`clawhub uninstall <slug>;`
- 或手动删除 ClawHub 技能目录下对应技能文件夹（目录名一般为 slug）。

### 各技能安装与卸载命令

上表「技能总表」中每个 Slug：**安装** `clawhub install <slug> --workdir ~/.openclaw;`，**卸载** `clawhub uninstall <slug>;`。示例：`clawhub install discord --workdir ~/.openclaw;` / `clawhub uninstall discord;`，`clawhub install discord-voice --workdir ~/.openclaw;` / `clawhub uninstall discord-voice;`；其余将 `<slug>` 替换为表中对应 Slug 即可。
