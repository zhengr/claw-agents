## 重点技能简介（Top 6）

### telegram · codedao12

用直接 HTTPS 请求（无 SDK）设计 Telegram Bot API 工作流与命令式对话，适合 OpenClaw 集成，高下载量与 Stars。

### telegram-api · byungkyu

Bot API 集成与托管认证：发消息、管理聊天、处理 updates、与用户互动。适用于发消息、创建投票、管理 bot 命令或与 Telegram 聊天互动时。其他第三方 app 可配合 api-gateway skill。

### telegram-bot · Sebastian-Buitrag0

通过 Telegram Bot API 构建与管理 Bot：创建 bot、发消息、配置 webhook、管理群组与频道。

### telegram-compose · tmchow

用 HTML 格式化并发送富文本消息；主会话对有实质内容的 Telegram 输出自动调用。规则：回复 >3 行或含列表/统计/章节/报告时，用 Haiku 子 agent 格式化发送；短回复直接走 OpenClaw message 工具。适用于研究摘要、告警、状态、报告、简报、通知等有视觉层级的内容。

### tg · arein

Telegram CLI：读、搜、发消息；用户要查收件箱、搜聊天、发消息、查联系人与群组时使用。

### telegram-auto-topic · itstauq

在论坛群组中任意消息前加 `/topic` 自动创建新话题，标题根据内容自动生成。Github: itstauq/telegram-auto-topic。

## 安装、更新与卸载

### 安装
- **搜索**：[ClawHub telegram](https://clawhub.ai/skills?sort=downloads&q=telegram) 查看最新技能与下载量。
- **命令**：`clawhub install <slug>`
- 安装后技能目录名需与 openclaw 配置中 `agents.list[].skills` 一致。
- **Token**：使用前配置 `BOT_TOKEN`（[@BotFather](https://t.me/BotFather) 创建 Bot），勿写入 TOOLS.md 或配置文件。
- **补充**：skills.sh 安装：`npx skills add <owner/repo> --skill <名>`；Telegram 技能列表见本目录 [README.md](./README.md)。

### 更新
- 重新执行 `clawhub install <slug>` 覆盖安装即可；具体以 ClawHub CLI 文档为准。

### 卸载
- **命令**：`clawhub uninstall <slug>`
- 或手动删除 ClawHub 技能目录下对应技能文件夹（目录名一般为 slug）。

### 各技能安装与卸载命令

上表「技能总表」中每个 Slug：**安装** `clawhub install <slug>`，**卸载** `clawhub uninstall <slug>`。示例：`clawhub install telegram` / `clawhub uninstall telegram`，`clawhub install telegram-bot-builder` / `clawhub uninstall telegram-bot-builder`；其余将 `<slug>` 替换为表中对应 Slug 即可。
